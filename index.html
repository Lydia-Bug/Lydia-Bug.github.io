MP Insight

Created in 2023 for COSC345

The requirements of this paper was to create a C++ windows app that made use of two datasets. I worked in a team of four, over the course of a semester. Our team decided to create an app that made use of data about current MP’s, to help inform the public about them, particularly coming up to the 2023 election. We created the Gui using the Qt framework, used a SQLite database, and python to scrape the data we collected from the parliament website.

Cafe queue simulation

Created in 2020 for NCEA level 3 Computer Science

A simulation of a queue at my high school cafeteria. The programme will ‘do something’ for every second in an hour, and it will remove people when they have been served, and add people as time goes by. This was the first large project I coded from scratch in java. The NCEA standard required the use of various complex techniques, I used a linked-list data structure to represent a queue, and the programme is able to read data from other files.
